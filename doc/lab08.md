</p>***Вміст файлу "main.c"***

Головний файл

Це файл, який містить точку входу функйію main, виклики функцій lab05, lab06 та
значення для аргументів цих функцій.

</p>`main(int argc, char *argv[])`

Головна функція.<br>

Аргументи

int argc, argv *c[] Аргументі які зберігають значення введені через командний рядок

argc - зберігаеє кількість введених значень

argv - зберігае самі значення<br>

Послідовність дій

1. Присвоїти значення аргументам argc і argv.

    * argc - аргумент типу int необхідний для обчислення всіх
щасливих квитків.

    * argv - масив типу char який зберігатиме в собі значення від
якого почнеться перевірка всіх квитків.

2. Створення змінних яким буде надано значення для аргументів
функцій link lab05, lab06.

    * t - зберігає значення за умовчуванням від якого буде
починатися перевірка квитків для функції lab06. Воно буде
використовуватися якщо не будуть передані аргументи
командного рядка.

    * x - зберігає рандомне значення, яке є мінімальним числом в
діапазоні для функції lab05.

    * y - зберігає рандомне значення, яке є максимальним числом в
діапазоні для функції lab05.

    * *c - змінна яка використовується для перевірки аргументів
командного рядка.

3. Гненеруємо рандомні числа за допомогою генератора rand() та
функції srand(), після чого привласнюємо їх пріменним x і y.

4. Потім викликаємо функцію lab05 і присвоюємо її аргументам
значення примінних x та y.

    srand((unsigned int)time(NULL));

    x = rand() % 11;

    y = rand() % (x-1);

    lab05(x, y);

5. Робимо перевірку на те, чи були введені аргументи через командний
рядок.

6. Якщо перевірка була пройдена, то переобразуємо аргумент
командного рядка в значення типу int і присвоюємо його примінній t

7. Робимо перевірку, чи не менше t мінімального значення квитка
1000.

8. Якщо перевірка була пройдена то присвоюємо t аргументам функції
lab06

9. Якщо одна з перевірок не була пройдена, то присвоюємо аргументам
функції lab06 значення за замовчуванням.


    if(t > 1000 && t < 9999)

    {

    lab06(t);

    }else

    {

    t= 1000;

    lab06(t);

    }

    }else

    {

    t= 1000;

    lab06(t);

    }

</p>`int lab05(a, b)`

Ця функція знаходить найбільше просте число в рандомному діапазоні.<br>

Аргументи

a, b - Діапазон чисел у якому відбувається пошук найбільшого числа.

а - верхнє число діапазону

b - нижня кількість діапазону
<br>

Послідовність дій

1. Створення змінних i, max.
    * i - використовується для перевірки просте число чи ні.

    * max - зберігає найбільше просте число.

* Запуск циклу для перевірки чисел у встановленому діапазоні.

    while(a > b)

* Запуск циклу для того, щоб дізнатися число просте чи ні.

    while(b%i != 0){

    i++;

    }

* Робимо перевірку щоб дізнатися число виявилося простим чи ні.
Якщо воно просте то перевіряємо чи більше воно числа яке зараз
зберігається в змінній max, якщо перевірка була пройдена то
присвоюємо змінної max це число.

    if(b == i && b > max){

    max = b;

    }

</p>`int lab06(k)`

Ця функція знаходить усі щасливі квитки в діапазоні від k до 9999.<br>

Аргументи

k - число від якого починається пошук квитків<br>

Послідовність дій

1. Створення змінних A[9999],sum1,sum2,n1,n2,n3,n4,n,i.
    * A - масив в який записуватимемо щасливі квитки.

    * Задаємо змінні sum1 і sum2 в яких буде
зберігатися суми першої і другої пари чисел.

    * Змінні n1, n2, n3, n4 зберігатимуть по одному
числу з номера квитка.

    * Змінна n – це кількість щасливих квитків.

2. Створення циклу який перевіряє всі квитки від k до 9999.

    while( k <= 9999)

3. Розбиваємо номер квитка на 4 числа.

    n1 = k/1000;

    n2 = (k/100)%10;

    n3 = (k/10)%10;

    n4 = k%10;

4. Дізнаємось суму першої пари чисел і другої.

    sum1 = n1 + n2;

    sum2 = n3 + n4;

5. Якщо сума першої пари дорівнює сумі другої, то записуємо
цей квиток у масив і додаємо 1 до кількості щасливих
квитків(n).

    if(sum1 == sum2){

    A[n] = k;

    n++;

    }

6. І збільшуємо номер квитка на +1.

    k++;
